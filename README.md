# INM5151-Class_Meet
Dépôt pour notre projet de session dans le cours INM5151 de la session A-21


## Comment lancer le projet
1. Installer [node.js](https://nodejs.org/en/)
2. Faites la commande `npm i`
3. Puis `npm run serve`
